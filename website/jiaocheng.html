方法一、使用外网IP

1.首先你需要自己的外网IP，如果不知道请百度“ip”。如图。

配置好网络的设置

2.打开CMD（Win+R，输入CMD，回车），输入ipconfig，然后回车，你会看到你的内网IP。

注意：是“以太网适配器”那个！！！

3.然后用浏览器打开你路由器的管理面板（通常是192.168.0.1），这里以腾达路由器为例。

4.点击更多功能→瑞口映射，按照上图填写

5.然后再找到“DMZ主机”，点击“启用”，主机IP地址填你的内网IP。

6.最后将UPnP打开，然后保存。

开始开服

1.用记事本或notepad++打开server.properties这个文件。找到server-ip这一行。

填写好你的内网IP。上面的server-port随便填（最好是20000-25565以内的数字）

2.在有“xx.1.xx.x-.jar”的目录下创建一个txt文件。

3.打开这个txt文件，写上一行：

java -Xincgc -Xms1g -Xmx1g -jar <xx.1.xx.x-.jar>.jar

4.保存这个txt文件，后缀改成.bat

5.打开这个bat文件，启动服务器。

出现“Done (70.100s)! For help, type "help" or "?"”之后就说明你的服务器启动成功了。

打开Minecraft→多人游戏→添加服务器，地址填 你的外网或内网IP:瑞口号

如果能进入游戏，那么恭喜你，你成功了。

注意：如果内网IP经常出现变动而导致无法正常启动服务器，请参见家用电脑映射开服完整详细图文教程- 联机教程- Minecraft(我的世界)中文论坛- 手机版- Powered by Discuz! 中的七。

外网进入你的服务器只需填写 你的外网IP:瑞口号

注意：如果是用ADSL上网，那么你的外网IP每天都会有变化！

内网IP有时也会有变化！所以你每隔一段时间都需要重复上面的步骤！
