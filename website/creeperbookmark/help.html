如何导入？

- 下载

  1. 首先点击这里，进入下载页面.
  2. 下载完成后，你会得到一个名为“bookmarks-202x-xx-xx.7z”的文件.
  3. 然后请解压这个文件，得到bookmarks-202x-xx-xx.html

- 导入

Chrome Firefox 360安全浏览器
IE浏览器 Safari Edge（新版）

这里没有你正在使用的浏览器？请百度一下.

-- Chrome

  1. 点击右上角的三个点->书签->导入书签和设置
  2. 选择以前导入的书签（HTML文件），然后点击导入
  3. 选择你所下载的书签文件
  4. 导入完成后，查看书签栏中是否出现“已导入”或“其他书签”文件夹，若出现即代表导入成功.

没看懂？点我前往谷歌官方文档查看。（需要魔法）

-- Firefox

  1. 点击右上角的三个横线->我的足迹->书签->管理书签->导入和备份.
  2. 选择从 HTML 文件导入书签
  3. 选择你所下载的书签文件
  4. 导入完成后，查看书签菜单文件夹中是否出现“书签栏”或“导航网站”等文件夹，若出现即代表导入成功.

没看懂？点我前往Firefox官方文档查看。

-- 360安全浏览器

  1. 点击收藏右侧的下拉按钮，选择导入/导出.
  2. 选择从HTML文件导入
  3. 选择你所下载的书签文件
  4. 导入完成后，查看书签菜单文件夹中是否出现“已导入”或“其他书签”等文件夹，若出现即代表导入成功.

没看懂？点我前往360官方文档查看。

-- IE浏览器

⚠ 注意！这里以IE11为例，在其他版本的IE可能不适用.

  1. 点击右上的星星⭐按钮，然后点击“添加到收藏夹”右侧的下拉按钮->导入和导出.
  2. 选择从文件导入，然后在下一步中选择“收藏夹”，再点击“浏览”
  3. 选择你所下载的书签文件
  4. 导入完成后，查看书签菜单文件夹中是否出现“已导入”或“其他书签”等文件夹，若出现即代表导入成功.

-- Safari

ℹ 提示：这段文字来自于Apple支持.

  1. 在 Mac 上的 Safari 浏览器应用中，选取“文件”>“导入自”>“书签 HTML 文件”。
  2. 选择要导入的文件。 导入书签后，它们将显示在边栏底部，位于名称以“已导入”开头并以日期结尾的新文件夹中。
  3. 点按“导入”。

没看懂？点我前往苹果官方文档查看。

-- Edge（新版）

⚠ 注意：此方法只适用于Edge的新版本.

  1. 点击右上角的书签图标->三个点->导入收藏夹
  2. 下拉导入位置选项，选择“收藏夹或书签 HTML 文件”
  3. 然后点击“选择文件”
  4. 选择你所下载的书签文件
  5. 导入完成后，查看书签栏中是否出现“已导入”或“其他书签”文件夹，若出现即代表导入成功.

没看懂？点我前往微软官方文档查看。
